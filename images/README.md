# Images Directory

## Purpose
This directory contains image assets for the CMU-Africa website replica. These images are used throughout the website to provide visual content including logos, banners, faculty photos, and other graphical elements.

## Structure
The images are organized to support the various sections of the CMU-Africa website:

- **Logos and Branding**: CMU and CMU-Africa institutional logos
- **Hero Images**: Banner and header images for the main pages
- **Faculty Photos**: Professional headshots and team photos
- **Program Images**: Visual content related to academic programs
- **Campus Photos**: Images showcasing the CMU-Africa campus and facilities
- **Event Images**: Photos from conferences, graduations, and other events

## Usage
These images are referenced in the HTML files and styled through CSS. All images should be optimized for web use to ensure fast loading times.

## File Formats
- **JPEG**: For photographs and complex images
- **PNG**: For logos and images requiring transparency
- **SVG**: For scalable vector graphics when available

## Guidelines
- Maintain consistent image quality and sizing
- Use descriptive filenames for better organization
- Optimize images for web performance
- Ensure proper attribution for any third-party images

## Testing
When adding new images:
1. Verify images display correctly across different screen sizes
2. Check loading performance impact
3. Ensure accessibility with proper alt text in HTML
4. Test on various browsers and devices
